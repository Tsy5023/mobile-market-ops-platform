# 指标树配置 - 功能设计（参考外部系统拆解）

> 页面路径：基础功能 → 配置管理 → **指标树配置**  
> 列表：`pages/indicator-tree.html`  
> 创建/编辑画布：`pages/indicator-tree-editor.html`

## 一、功能模块总览

| 模块 | 说明 | 本系统页面 |
|------|------|------------|
| 指标树列表 | 查询、筛选、分页、生命周期操作 | `indicator-tree.html` |
| 指标树创建/编辑 | 画布搭建上下游节点 + 侧栏配置 | `indicator-tree-editor.html` |
| 指标树详情（只读） | 查看结构与元数据 | 编辑器 `?view=1` |
| 指标洞察下钻 | 发布后运营侧分析 | `indicator-tree-detail.html`（能力中心） |

## 二、列表页功能拆解（参考图 1、2）

### 2.1 筛选区

| 字段 | 类型 | 说明 |

| 归属区域 | 下拉 | 全国 / 江西省 等 |

| 业务场景 | 下拉 | 按场景路径筛选 |
| 状态 | 下拉 | 草稿 / 发布中 / 已发布 / 已下线 |

操作：**展开/收起**、**重置**、**搜索**

（参考图 2 扩展列：修改人、修改时间、有效期 — 可在详情/编辑中维护，列表原型以核心列为主。）

### 2.3 行操作（按状态）

| 状态 | 操作 |
|------|------|
| 发布中 | 详情、复制 |
| 已发布 | 详情、编辑、下线、复制 |

- **分页**：总条数、每页条数、页码、跳转

## 三、画布创建/编辑页（参考图 3～6）

### 3.1 顶栏

| 按钮 | 说明 |
|------|------|
| 取消 | 返回列表，未保存提示 |
| 保存草稿 | 状态=草稿 |
| 预览 | 跳转洞察/下钻预览（演示） |
| 发布 | 校验后提交发布 |

### 3.2 画布区

| 能力 | 说明 |
|------|------|
| 点阵背景 | 对齐参考系统画布 |
| 横向树结构 | 左→右表达上下游/父子关系 |
| 节点卡片 | 名称展示；选中高亮（蓝框） |
| 连线 | 节点间水平箭头连线 |
| 添加子节点 | 卡片右侧「+」 |
| 删除节点 | 卡片右侧「−」（根节点不可删） |
| 缩放工具条 | 放大/缩小/重置 |

### 3.3 右侧栏 - 基本信息（未选节点或编辑根元数据）

| 字段 | 必填 | 说明 |
|------|------|------|
| 指标树名称 | 是 | |
| 归属区域 | | |
| 分析粒度 | | |
| 业务场景 | | 多级场景路径 |
| 指标树描述 | | 500 字计数 |
| 同时设置为模板 | | 复选框 |

### 3.4 右侧栏 - 节点属性（选中画布节点）

| 字段 | 节点类型 | 说明 |
|------|----------|------|
| 节点名称 | 全部 | 必填 |
| 节点类型 | 全部 | 分类节点 / 业务指标 |
| 指标 | 业务指标 | 弹窗检索选择单条指标（编码、名称、场景等） |

**分类节点**：仅名称 + 类型，用于结构分组。  
**业务指标**：绑定指标库中的单个指标，作为节点展示与下钻的数据来源。

## 四、与指标洞察的关系

- **配置侧**（本模块）：定义树结构、节点指标、版本与发布状态。
- **洞察侧**（指标洞察/详情）：消费已发布指标树，做下钻、根因、预警分析。
- 列表「详情」只读打开编辑器；运营分析走 `indicator-tree-detail.html`。

## 五、数据模型（演示）

```text
IndicatorTreeConfig
├── code, name, scene, status, version
├── region, granularity
├── creator, createdAt, modifier, modifiedAt
└── graph: { meta, nodes: TreeNode }

TreeNode
├── id, name, type (category | metric)
├── metricId, metricCode, metricName
└── children: TreeNode[]
```

演示数据：`assets/js/indicator-tree-config-data.js`
