## 2. 需求背景

### 2.1 业务背景

江西移动存量与流量运营中，营销策略来源多样：既有面向特定客群的常规营销，也有需网格/厅店线下执行的工单型任务，还有由指标异常触发的预警改善策略。当前策略分散在多个系统与表格中，运营人员难以：

- 按策略**类型**快速定位与管理；
- 统一查看策略**配置要点**与**执行闭环**（客群触达 / 工单下派 / 指标改善）；
- 从策略分类页一键跳转至**调度中心**或**指标树**继续操作。

### 2.2 产品目标

1. 建立**三类策略分类体系**（客群类、工单类、预警类），统一入口浏览与检索。
2. 采用**主从式交互**，左侧列表、右侧详情，降低切换成本。
3. 工单类策略突出**调度中心对接**与**线下网格执行**描述；预警类策略突出**根因分析**与**指标改善跟踪**。
4. 与策略推荐、策略创建、调度派发、指标树等模块形成跳转闭环。

### 2.3 目标用户

| 角色 | 典型诉求 |
|------|----------|
| 省公司策略运营 | 浏览全省三类策略清单，查看配置与执行状态 |
| 地市运营人员 | 检索本地相关策略，跳转调度中心处理工单 |
| 流量/存量专项运营 | 查看预警类策略及指标改善进展 |
| 数据分析人员 | 查看预警类策略的根因结论与关联指标 |

### 2.4 范围说明

**本期覆盖（V1.0 原型已实现）：**

- 策略分类页三 Tab 切换
- 策略列表检索与选中
- 三类策略差异化详情展示
- 跳转调度中心、指标树、策略推荐/创建

- 列表状态筛选、新增/编辑/启停策略弹窗
- 与后端 API 及权限体系对接

## 4. 功能详述

### 4.1 策略分类页

**页面路径：** `pages/strategy-category.html`  
**菜单路径：** 能力中心 → 策略中心 → 策略分类

#### 4.1.1 页面框架与导航

##### 4.1.1.1 功能详细说明

**用户可实现：** 进入策略中心统一入口，了解当前模块定位，并在三类策略之间切换浏览。

**页面组成：**

| 区域 | 元素 | 说明 |
|------|------|------|
| 顶部导航 | 一级模块 Tab、用户信息 | 平台通用，由 `layout.js` 渲染 |
| 侧边栏 | 策略中心二级菜单 | 含策略分类、策略推荐、策略统筹等 |
| 面包屑 | 首页 / 能力中心 / 策略中心 / 策略分类 | 可点击返回首页 |
| 页头 | 标题「策略分类」、副标题 | 说明「按客群、工单、预警三类管理策略体系」 |
| 类型 Tab 区 | 3 个 Tab 按钮 | 客群类 / 工单类 / 预警类 |
| 类型说明区 | 图标 + 文字描述 | 随 Tab 切换更新 |
| 主从内容区 | 左侧列表 + 右侧详情 | 见 4.1.2、4.1.3 |

**页面操作：**

| 操作 | 触发方式 | 结果 |
|------|----------|------|
| 切换一级模块 | 点击顶部导航 | 跳转对应模块首页 |
| 切换侧边菜单 | 点击左侧菜单项 | 跳转策略中心其他页面 |
| URL 带参进入 | `?tab=segment/workorder/alert` | 默认打开对应 Tab |

**交互规则：**

1. 页面加载默认选中「客群类策略」Tab；若 URL 含合法 `tab` 参数则打开对应 Tab。
2. 切换 Tab 时清空当前选中策略，重新加载该类型列表，并自动选中第一条（若有数据）。

#### 4.1.2 策略类型 Tab

##### 4.1.2.1 功能详细说明

**用户可实现：** 按业务类型划分浏览策略，快速理解各类型策略的能力边界与适用场景。

**Tab 定义：**

| Tab | 编码 | 功能描述（页面展示文案） |
|-----|------|--------------------------|
| 客群类策略 | `segment` | 面向特定客群的精准营销策略，支持客群圈选、产品推荐、渠道触达与效果跟踪，是策略中心的基础策略类型。 |
| 工单类策略 | `workorder` | 支持对接调度中心，基于策略配置封装调度中心任务，生成待执行的工单，将工单下派给指定执行对象，通过线下网格进行策略执行。 |
| 预警类策略 | `alert` | 支持基于指标溯源与根因分析智能生成预警类策略，精准定位短板问题，输出针对性策略，实现针对指标根因分析的策略应对，助力运营指标提升改善。 |

**页面操作：**

| 操作 | 触发 | 视觉反馈 |
|------|------|----------|
| 切换 Tab | 点击 Tab 按钮 | 当前 Tab 高亮（蓝底）；说明区文案与图标同步更新 |
| — | — | 左侧列表刷新为当前类型数据；右侧详情随列表选中项更新 |

**与数据状态关系：** Tab 切换不涉及策略状态，仅过滤策略 `category` 字段。

#### 4.1.3 策略列表（左侧面板）

##### 4.1.3.1 功能详细说明

**用户可实现：** 在当前策略类型下检索目标策略，通过卡片式列表快速浏览摘要，并选中某条策略查看完整详情。

**查询条件：**

| 字段 | 组件类型 | 说明 | 校验/规则 |
|------|----------|------|-----------|
| 关键词 | 单行输入框 | 占位符「搜索策略名称 / ID」 | 可选；对策略名称、策略 ID、策略子类、摘要做模糊匹配（不区分大小写） |
| 策略类型 | Tab 隐式条件 | 当前激活 Tab | 固定，不可在列表区单独修改 |
| 策略状态（扩展） | 下拉单选 | 全部 / 草稿 / 待启用 / 执行中 / 已暂停 / 已结束 | P1 规划，V1.0 原型未实现 |

**列表展示字段（卡片）：**

| 字段 | 展示位置 | 说明 |
|------|----------|------|
| 策略名称 | 卡片标题 | 加粗主标题 |
| 策略状态 | 卡片标题右侧 | 徽标：草稿(灰)、待启用(黄)、执行中(绿)、已暂停(蓝)、已结束(灰) |
| 策略 ID | 卡片元信息行 | 等宽字体 |
| 策略子类 | 卡片元信息行 | 如离网维系、指标改善等 |
| 策略摘要 | 卡片正文 | 最多展示 2 行，超出省略 |
| 责任部门 | 卡片底部 | 带用户图标 |
| 更新时间 | 卡片底部 | 格式 YYYY-MM-DD |

**列表统计：** 列表顶部展示「共 N 条」，随筛选结果实时更新。

**页面操作：**

| 操作 | 触发 | 结果 | 与状态关系 |
|------|------|------|------------|
| 搜索 | 点击搜索按钮或输入框回车 | 按关键词过滤列表 | 无关 |
| 选中策略 | 点击列表卡片 | 卡片高亮；右侧渲染对应详情 | 无关 |
| 自动选中 | Tab 切换或搜索后 | 若无选中项且有数据，默认选中第一条 | 无关 |
| 空态 | 无匹配数据 | 展示「暂无匹配策略」 | — |

**交互规则：**

1. 列表区域可纵向滚动，选中态以蓝色边框标识。
2. 切换 Tab 时搜索框内容保留，但列表数据按新 Tab + 关键词重新过滤。
3. 扩展：状态筛选与关键词为 AND 关系。

#### 4.1.4 策略详情（右侧面板 · 公共）

##### 4.1.4.1 功能详细说明

**用户可实现：** 查看所选策略的完整配置与业务说明，并根据策略类型执行跳转操作。

**公共展示规则：**

| 元素 | 说明 |
|------|------|
| 策略类型标签 | 图标 + 文案（客群类/工单类/预警类） |
| 策略名称 | H2 标题 |
| 策略 ID + 状态 | 副标题行，状态徽标与列表一致 |
| 策略摘要 | 一段完整业务描述 |

**空态：** 未选中任何策略时，展示「请从左侧选择一条策略查看详情」引导文案。

**页面操作（公共）：**

| 操作 | 可见条件 | 说明 |
|------|----------|------|
| 查看详情 | 选中任意策略 | 按 category 渲染不同详情模板 |
| — | 无选中 | 空态引导 |

#### 4.1.5 客群类策略详情

##### 4.1.5.1 功能详细说明

**用户可实现：** 查看面向特定客群的营销策略完整配置，包括圈客规则与执行步骤，并跳转至策略推荐列表或创建同类新策略。

**详情信息区（卡片网格）：**

| 字段 | 说明 |
|------|------|
| 策略子类 | 如离网维系、降档防控、加包增收、合约运营 |
| 责任部门 | 创建/归属组织 |
| 目标客群 | 客群名称 + 规模（如 4,280 人） |
| 推荐产品 | 主推产品名称 |
| 触达渠道 | 如外呼+短信、APP+厅店 |
| 覆盖地市 | 逗号分隔地市列表或「全省」 |
| 生效周期 | 起止日期 |
| 策略目标 | KPI 描述，如转化率目标 |

**详情区块：**

| 区块 | 内容 |
|------|------|
| 圈客规则 | 无序列表，展示标签/规则条件条目 |
| 执行步骤 | 有序步骤列表，带序号圆标 |

**页面操作：**

| 操作 | 按钮文案 | 触发结果 | 与状态关系 |
|------|----------|----------|------------|
| 跳转策略推荐 | 「策略推荐列表」 | 跳转 `strategy-recommend.html` | 无关 |
| 创建同类策略 | 「创建同类策略」 | 跳转 `strategy-create.html` | 建议草稿/待启用策略可编辑；执行中策略跳转时为「参照创建」 |

**扩展操作（P2 规划，与状态相关）：**

| 操作 | 可见状态 | 说明 |
|------|----------|------|
| 编辑 | 草稿、待启用 | 打开编辑弹窗 |
| 启用 | 待启用 | 状态变更为执行中 |
| 暂停 | 执行中 | 状态变更为已暂停 |
| 结束 | 执行中、已暂停 | 状态变更为已结束 |

| 生效周期 | 起止日期 |

**详情区块 · 调度闭环流程：**

| 展示 | 说明 |
|------|------|

**页面操作：**

#### 4.1.7 预警类策略详情

##### 4.1.7.1 功能详细说明

**用户可实现：** 查看由指标异常触发的预警类策略，了解关联指标、根因分析结论、针对性改善动作及指标改善进展，并跳转指标树进一步下钻分析。

**预警横幅：**

| 元素 | 说明 |
|------|------|
| 预警等级 | 红色/橙色/黄色预警，对应不同背景色 |
| 关联指标 | 指标名称 |
| 指标路径 | 如「收入类 > 套外收入 > 套外收入占比」 |

**详情信息区（卡片网格）：**

| 字段 | 说明 |
|------|------|
| 分析来源 | 如指标溯源+根因分析引擎、指标预警+多维下钻 |
| 改善目标 | 量化目标，如「套外收入占比下降 0.8pp（30 日内）」 |
| 覆盖范围 | 涉及地市 |
| 执行周期 | 起止日期 |
| 责任部门 | 归属组织 |
| 策略子类 | 如指标改善、结构优化、风险防控 |

**详情区块 · 根因分析结论：**

| 展示 | 说明 |
|------|------|
| 结论文案 | 左侧蓝色竖线强调，完整描述根因 |

**详情区块 · 针对性策略动作：**

| 展示 | 说明 |
|------|------|
| 动作卡片 | 每卡片含「动作标题 + 动作说明」，2 列网格布局 |

**详情区块 · 指标改善跟踪（子列表）：**

| 列名 | 说明 |
|------|------|

#### 4.1.8 新增策略弹窗（扩展需求）

##### 4.1.8.1 功能详细说明

**用户可实现：** 在当前 Tab 对应类型下快速新建一条策略，填写差异化配置并保存为草稿或提交启用。

**弹窗触发：**

| 入口 | 位置 |
|------|------|
| 新增按钮 | 列表工具栏右侧「+ 新增策略」 |

**公共表单字段：**

| 字段 | 组件类型 | 必填 | 校验规则 |
|------|----------|------|----------|
| 策略名称 | 单行文本 | 是 | 2～50 字；同类型下不可重名 |
| 策略子类 | 下拉单选 | 是 | 选项随 Tab 类型变化 |
| 责任部门 | 下拉/组织选择器 | 是 | 须选至部门级 |
| 生效开始日期 | 日期选择器 | 是 | ≥ 当前日期 |
| 生效结束日期 | 日期选择器 | 是 | > 开始日期 |
| 覆盖地市 | 多选下拉 | 是 | 至少 1 个；支持「全省」快捷 |
| 策略摘要 | 多行文本 | 是 | 10～500 字 |

**客群类 · 附加字段：**

| 字段 | 组件类型 | 必填 | 校验规则 |
|------|----------|------|----------|

| 推荐产品 | 下拉单选 | 是 | 来自产品库 |
| 触达渠道 | 多选 + 主次渠道 | 是 | 至少 1 个渠道 |
| 策略目标 | 多行文本 | 是 | KPI 描述 |
| 圈客规则 | 动态规则行 | 是 | 至少 1 条；支持且/或 |

**工单类 · 附加字段：**

| 字段 | 组件类型 | 必填 | 校验规则 |
|------|----------|------|----------|
| 调度任务模板 | 下拉单选 | 是 | 来自调度中心已注册模板 |
| 执行对象 | 下拉单选 | 是 | 网格经理/厅店经理/微格团队等 |
| 执行方式 | 单选 | 是 | 线下网格/厅店/外呼等 |
| SLA | 数字 + 单位下拉 | 是 | 正整数；单位：小时/天 |
| 触发条件 | 多行文本 / 规则编辑器 | 是 | 须可解析为调度触发表达式 |
| 工单类型 | 下拉 | 是 | 与调度中心工单类型字典一致 |
| 优先级 | 单选 | 是 | 高/中/低 |

**预警类 · 附加字段：**

| 字段 | 组件类型 | 必填 | 校验规则 |
|------|----------|------|----------|
| 关联指标 | 指标树选择器 | 是 | 须选叶子指标 |
| 预警等级 | 单选 | 是 | 红/橙/黄 |
| 分析来源 | 多选 | 是 | 溯源/根因/竞品等 |
| 改善目标 | 单行文本 | 是 | 须含量化目标 |
| 根因结论 | 多行文本 | 是 | 20～1000 字 |
| 策略动作 | 可增删卡片组 | 是 | 至少 1 条；每条含标题+说明 |
| 跟踪指标 | 可增删表格行 | 是 | 至少 1 个；含指标名、基线、目标 |

**弹窗底部操作：**

| 按钮 | 行为 | 与校验 |
|------|------|--------|
| 取消 | 关闭弹窗，不保存 | — |

**错误提示：** 字段级红色文案 + 弹窗顶部汇总（若有 ≥3 个错误）。
