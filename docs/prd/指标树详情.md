## 1. 洞察中心

### 1.1 指标树详情

> 页面说明：以横向树形式展示某一指标树的分类与指标节点，支持展开收起、检索定位与画布平移缩放，用于日常异动巡检与下钻分析入口。  
> 页面文件：`pages/indicator-tree-detail.html`  
> 入口：指标洞察列表点击某棵指标树进入（参数：`?id=` 指标树 ID）

---

#### 1.1.1 进入页面默认展示规则

**功能1：树数据加载** （功能利旧）

- 根据 URL 中的指标树 ID 加载对应树根节点及完整子树（来源：指标树详情数据）
- 面包屑展示当前树名称（来源：指标树配置 `name`）
- 空树 / 加载失败：【待补充：正式环境错误提示文案】

**功能2：默认展开深度**（本次调整功能）

- 进入页面后，默认按**展开到第 2 层**处理（根为第 0 层）
- 展开规则（「深度展开 + 分类连展」）：
  1. 先按目标深度展开：深度小于目标深度且存在子节点的节点均展开
  2. 到达目标深度后，若该节点为**分类节点**且子节点**全部仍是分类**，则继续向下自动展开
  3. 直到某一层出现**至少一个指标节点**为止，该层不再继续连展
- 目的：避免用户连续点击空分类，尽快看到可分析的指标层

**功能3：全部收起后的保底态**

- 「全部收起」后仍保留根节点展开，确保画布上至少可见第一层子节点

**功能4：页面提示条（可选展示）**

- 展示业务提示（如：异动识别范围说明、节点变更后报告再生时效）
- 用户可关闭；关闭状态本地记忆（来源：浏览器本地存储）

#### 1.1.2 指标树布局规范

**功能1：整体布局形态**（调整功能）

| 项 | 说明 |
| --- | --- |
| 布局方向 | 横向树：父节点在左，子节点向右展开 |
| 对齐方式 | 顶对齐；子节点纵向排列，减少中间大面积空白 |
| 连线 | SVG 正交折线：父卡片右中点 → 总线 → 子卡片左中点 |
| 画布工具栏 | 悬浮于画布上方，不随拖拽平移（左侧展开控制 / 右侧视图控制） |

**功能2：节点类型与卡片内容**（调整功能）

| 节点类型 | 展示内容 | 视觉区分 | 数据来源 |
| --- | --- | --- | --- |
| 根节点 | 节点名称 | 相对放大强调 | 指标树根配置 |
| 分类节点 | 文件夹图标 + 分类名称（可多行省略） | 窄卡片、分类样式 | 树结构 `type=category` |
| 指标节点 | 名称、业务口径图标、指标值、环比 | 常规指标卡片 | 指标库 / 树节点快照 |
| 异动指标 | 同上 | **仅红色边框**（指标值保持常规色） | `alert` / `status=alert` |

**功能3：指标卡片字段**

| 字段 | 说明 | 数据来源 |
| --- | --- | --- |
| 指标名称 | 最多约 2 行省略，悬停可看全称 | 节点 `name` |
| 业务口径图标 | 小号信息图标；悬停展示口径说明 | 节点 `caliber` |
| 指标值 | 当前展示值 | 节点 `value`（可按账期/地域演示微调） |
| 环比 | 展示「环±x」；正负色区分 | 节点 `mom` |

**功能4：展开连接器（收起态计数）**（调整功能）

- 有子节点的节点，卡片右侧延伸一小段横线，再接展开按钮（避免大数字遮挡连线）
- **收起态**显示计数：`X/YY`
  - `YY`：该节点下游指标总数（含各层指标；分类本身不计入）
  - `X`：其中异动指标数，**红色**强调
- 悬停提示：`包含YY个指标，其中有X个异动指标`
- **展开态**：计数隐藏，改为收起箭头；悬停提示「收起子指标」

**功能5：节点尺寸与层级强调**（功能调整）

- 仅**根节点**的卡片适当放大；一级分类（如「一个目标 / 三条主线」）不额外放大
- 分类卡片宽度较窄，长文案允许换行（约最多 4 行）

#### 1.1.3 画布展开控制

> 工具栏左侧：全部展开 / 全部收起 / 快速展开 / 指标检索

**功能1：全部展开**（利旧）

- 将所有含有子节点的节点加入展开集合并重绘
- 连线与布局随之全量刷新

**功能2：全部收起**（利旧）

- 清空展开集合，仅保留根节点展开
- 重绘后回到「根 + 第一层」可见态

**功能3：快速展开**（新增功能）

- 入口：按钮「快速展开」，支持悬停 / 点击展开菜单
- 菜单项按当前树**最大深度**动态生成：`展开一级` … `展开N级`
- 选择「展开 N 级」后，执行与默认进入相同的**深度展开 + 分类连展**规则（目标深度 = N）
- 无有效深度时隐藏快速展开入口

**功能4：节点级展开 / 收起**（利旧）

- 点击节点右侧展开按钮：
  - 若已展开 → 收起该节点（子树从画布移除）
  - 若未展开 → 执行「分类连展至出现指标」后再展开，减少空分类连点
- 收起仅影响当前节点展开状态，不强制收起兄弟节点

---

#### 1.1.4 指标检索

**功能1：打开检索面板**（新增功能）

- 入口：画布工具栏「指标检索」
- 展开输入框与结果列表；与「快速展开」菜单互斥关闭

**功能2：模糊查询**（新增功能）

| 项 | 说明 |
| --- | --- |
| 匹配范围 | 节点名称；同时可匹配路径文案 |
| 匹配方式 | 模糊查询 |
| 排序 | 完全匹配 > 前缀匹配 > 包含匹配；同级按名称排序 |
| 空输入 | 提示「输入关键词，模糊匹配指标或分类」 |
| 无结果 | 提示「无匹配的指标或分类」 |

**功能3：结果展示与加载**（新增功能）

| 字段/元素 | 说明 |
| --- | --- |
| 类型图标 | 文件夹 = 分类；折线 = 指标 |
| 异动图标 | 异动指标额外展示警示图标 |
| 名称 | 节点名称 |
| 路径 | 祖先路径摘要（过长省略） |
| 结果计数 | 「共 N 条结果」 |
| 分页加载 | 首屏约 20 条；列表滚动到底继续加载 |

**功能4：点击跳转**（新增功能）

1. 沿根到目标节点路径，自动展开所有祖先节点（未展开则代为展开）
2. 关闭检索面板并重绘画布
3. 将目标节点平移至画布可视中心附近
4. 目标卡片短暂高亮闪烁，便于确认定位

---

#### 1.1.5 画布视图操作

> 工具栏右侧顺序：全屏 → 回到初始位置 → 放大 → 缩小 → 缩放比例文案

**功能1：拖拽平移**（新增功能）

- 在画布空白区域按住拖动，平移整棵树
- 拖拽过程中不触发节点点击；忽略节点卡片、展开按钮、工具栏、输入控件上的拖拽起点
- 工具栏位置固定，不随平移移动

**功能2：放大 / 缩小**（新增功能）

| 项 | 说明 |
| --- | --- |
| 放大 | 步进约 +10% |
| 缩小 | 步进约 -10% |
| 范围 | 约 60% ~ 150% |
| 比例展示 | 工具栏显示当前百分比（如 100%） |
| 缩放锚点 | 以画布中心为参考缩放，尽量减少视口跳动 |

**功能3：全屏**（新增功能）

- 进入：指标树舞台区域全屏（优先系统全屏 API，失败则页面内全屏样式兜底）
- 退出：再次点击或退出系统全屏；按钮图标在进入/退出态切换

**功能4：回到初始位置**（新增功能）

- 重置缩放为 100%
- 重置平移到默认偏移（进入页的初始视口位置）
- 重算连线布局

## 2. 交互逻辑汇总

### 2.1 进入页主路径

1. 用户从指标洞察进入 `indicator-tree-detail.html?id=xxx`
2. 系统加载树数据并渲染横向树
3. 执行「展开到第 2 层 + 分类连展至出现指标」
4. 初始化画布平移/缩放，绘制父子连线
5. 用户可继续使用展开、检索、视图操作巡检节点

### 2.2 展开规则复用关系

| 场景 | 是否使用「深度展开」 | 是否使用「分类连展」 |
| --- | --- | --- |
| 进入页默认展示 | 是（深度=2） | 是 |
| 快速展开 N 级 | 是（深度=N） | 是 |
| 点击节点展开按钮 | 否（仅当前节点） | 是 |
| 全部展开 | 全量展开 | 不适用 |
| 全部收起 | 仅保留根展开 | 不适用 |
| 检索跳转 | 展开路径祖先 | 不强制连展兄弟分支 |

### 2.3 画布重绘时机

以下操作后均需重绘节点并重算连线：

- 展开 / 收起（含全部、快速、节点级）
- 检索跳转导致的祖先展开
- 缩放、复位、全屏尺寸变化、窗口 resize

### 4.2 名词说明

| 名词 | 说明 |
| --- | --- |
| 分类节点 | 仅组织树结构，不承载指标值 |
| 指标节点 | 可展示值与环比，可进入分析/调度 |
| 异动指标 | 当前账期下被识别为异常的指标节点 |
| 分类连展 | 连续展开纯分类层级，直到出现指标节点 |
| 深度 | 根为 0；「展开二级」表示展开到 depth&lt;2 的节点 |
