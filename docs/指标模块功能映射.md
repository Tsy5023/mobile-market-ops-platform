# 指标模块功能映射（来自 Excel 功能清单）

## 指标洞察（5 个功能点 → 页面能力）

**列表页** `indicator-insight.html`：指标树查询/新增、分类 Tab（全部/中高端/离网/家庭/国际）、卡片列表（名称/分类/数量/更新时间/异常状态）、详情/编辑。

**详情页** `indicator-tree-detail.html`：横向指标树（节点展示名称/值/同比/环比、口径悬停、正常异常图标、逐层展开收起）、点击节点打开分析弹窗。

| Excel 功能点 | 页面实现 |
|-------------|----------|
| 指标树可视化与下钻分析 | 详情页横向树 + 地域下钻 + 逐层展开 |
| 实时监控与趋势分析 | 弹窗「指标趋势」日周月年 + 同环比目标对比 |
| 指标异动告警与定位 | 弹窗「预警分析」规则/定位/告警记录 |
| 多维度指标分析 | 弹窗「多维分析」区域渠道等产品客户分层 |
| AI 根因分析 | 弹窗「根因分析」报告 + 客群关联 + 建议 |

## 指标配置（5 个功能点 → 页面能力）

| Excel 功能点 | 功能点描述（摘要） | 页面实现 |
|-------------|-------------------|----------|
| 数据同步 | 经分、IOP 等同步指标/客群/标签 | 「数据同步」页签：对接列表、任务监控、日志 |
| 基础信息配置 | 指标定义、口径、正负向属性 | 新建/编辑弹窗：编码、名称、口径、正负向、业务域等 |
| 数据源绑定 | 多数据源关联 | 编辑弹窗「数据源绑定」：多行绑定与优先级 |
| 审批发布与权限管控 | 审批后生效；角色层级权限 | 「审批发布」页签 + 权限说明；提交/审批流程 |
| 指标库查询与维护 | 多维度查询；全生命周期 | 「指标库」：筛选、状态流转、列表维护 |

## 指标树配置（4 个功能点 → 页面能力）

| Excel 功能点 | 功能点描述（摘要） | 页面实现 |
|-------------|-------------------|----------|
| 拖拽式指标树构建 | 父子关系、场景搭建 | **列表** `indicator-tree.html` + **画布** `indicator-tree-editor.html`：横向树节点、+/- 增删子节点、分类/业务指标节点属性 |
| 指标与客群关联 | 指标—客群影响关系 | 详情弹窗「关联客群」Tab 展示名称、规模、画像 |
| 模型训练与版本管理 | 版本管理、发布 | 列表版本列、状态（草稿/发布中/已发布）、保存草稿/发布 |
| 图形化展示与下钻 | 图形化展示、节点详情 | 画布图形化编辑；发布后由指标洞察详情页下钻分析 |

详见 `docs/指标树配置功能设计.md`。
